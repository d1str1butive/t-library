консольное приложение на C#
функционал:
добавление книг, сохранение информации о названии, авторе, жанре, годе издания и кратком описании
просмотр библиотеки с возможностью сортировки
удаление книг
поиск по ключевым словам (поиск идёт по названию, автору и описанию)
отметки о прочтении
добавление книг в избранное и удаление оттуда
все изменения автоматически сохраняются в library.json, загружаются при запуске
